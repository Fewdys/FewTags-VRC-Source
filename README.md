## Example Of How Local Tags Text File Should Look
```
Example 1: Multiple Tags On A User
Tag1,Tag2,<color=#ff0000>Tag3</color>:SomeUserID

Example 2: A UserID Example
<color=#ff0000>Red Tag</color>:usr_00000000-0000-0000-0000-000000000000

Example 3: Multiple Users With Tags
<color=#ff0000>Red Tag</color>:usr_00000000-0000-0000-0000-000000000000
<color=#ff0000>Red Tag</color>,<color=#00FFFF>Cyan Tag</color>:usr_00000000-0000-0000-0000-000000000001

Example 4: Animated Tags
.AnimatedPrefix.TagText:usr_00000000-0000-0000-0000-000000000000

Example 5: Use Of Rainbow Animated Tag and Blink Animated Tag
.RAIN.This Is A Rainbow Tag,.BLINK.Is This Thing Blinking?:usr_00000000-0000-0000-0000-000000000000
```

## List Of Animated Tags:
```
Scroll: Scrolls Text To The Left On Loop
CYLN: Bounces A Red Line Across Text Back And Forth (Supports One Color For Coloring)
LBL (Letter By Letter): Removes Letters/Characters One By One Than Adds Them Back One By One (No Support For Coloring)
RAIN (Rainbow): Old Call Of Duty World At War Modded Clan Tag Like Rainbow (No Support For Coloring)
SR (Smooth Rainbow): You're Generic Smooth Rainbow Animation (No Support For Coloring)
Pulse: Wave Like Text Animation Where Letters/Characters Get Bigger As It Goes Through Text Than Go Back To Normal Size Once Passed (Supports Coloring)
Jump: What Better Way To Describe This Other Than The Fact That Text Jumps In A Smooth Way (Text Goes Up And Down Creating A Jump Like Effect) (Supports Coloring)
Shake: Shake Text From Side To Side (Partly Supports Coloring)
GT (Ghost Trail): A Black Fade Goes Across Text I Suppose Would Be The Best Way To Explain It (No Support For Coloring)
Blink: Blinks Text (No Support For Coloring)
Glitch: Random Glitch Effect Throughout Text (Partly Supports Coloring -- Sorta)
```
